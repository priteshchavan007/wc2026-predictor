Firebase Migration: Multi-Group Support

This script migrates existing data into the new groups/default structure.

New structure:

  groups/
    default/
      info: { name, code, createdBy, createdAt }
      members: { phone: { role, joinedAt } }
      players: { phone: { name, phone, joined } }
      predictions: { phone_matchId: pick }
  results/ (global - unchanged)
  users/ (global user profiles)
  groupCodes/ (code -> groupId lookup)
